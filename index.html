Movies App

A non-trivial learning app in Next.js or Nuxt.js

The Movies App is a non-trivial demo application built on top of the TMDB (The Movie Database) API. It features multiple nested routes and views, authentication, dark mode and a large range of performance optimizations. The goal here is to educate developers on how they can build a good user-experience that can both help on the Core Web Vitals while maintaining good DX.

Views include:

  • Popular Movies - with pagination
  • Individual Movies - with recommendations, cast, and more.
  • Cast member/Actor with history
  • Categories - popular, TV/Movies and more.
  • Search - find, filter, sort movies. Paginated.
  • Authentication with TMDB - login, logout.
  • Lists - create new lists
  • Lists - view all your saved lists
  • Lists - add/remove/edit lists
